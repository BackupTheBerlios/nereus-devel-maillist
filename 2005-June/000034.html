möglicher Weise hatte ich den UnkooperativerBDIAgent von Eugen
fälschlicher Weise verdächtigt, falsch zu fliegen. Der Fehler tritt
nämlich immer noch beim EinfacherAgent auf. Wie ich nun gesehen habe,
findet sich in dessen Quelltext folgender Abschnitt (Zeile 172-179):

>>>
        if (id == 3) {
            fliegen(3, 1);
            landen();
            for(i = 0; i < 20; i++) {
                zuschauen();
            }
            starten();
        }
<<<

Offensichtlich fliegt der Agent (in Abhängigkeit von seiner ID!)
unabhängig von seiner aktuellen Position zur Position (3, 1).

Ist es sinnvoll, zu versuchen, diesen Agenten zu korrigieren?
